## AI Automation Learning Compass

Curated dashboard of the highest-signal courses, toolkits, and communities for mastering AI automation. Filter learning paths by skill level, format, and focus area while following a four-phase roadmap that keeps you shipping value as you learn.

### Features

- Searchable catalog of agentic AI and automation programs with highlights, skills, and ROI context.
- Filter controls for skill level, format, and focus area to surface the right course for your next milestone.
- Learning roadmap aligning discovery, prototyping, hardening, and scaling phases with recommended programs.
- Resource deck of newsletters, templates, tool indexes, and communities to sustain momentum.
- Responsive, Vercel-ready UI built with Next.js App Router and Tailwind CSS.

### Running Locally

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to explore the guide.

### Production Build

```bash
npm run build
npm run start
```
